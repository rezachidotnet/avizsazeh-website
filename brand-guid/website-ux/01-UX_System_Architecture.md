# 1. تعریف سیستم

## AECS UX System چیست؟

AECS UX System یک معماری تجربه کاربری است که به جای طراحی وبسایت، یک:

Architectural Engineering Interaction System

برای نمایش، درک و تبدیل پروژه‌های سقف و سیستم‌های معماری طراحی می‌کند.

---

# 2. هدف اصلی سیستم

1. تبدیل برند به System Thinking Entity  
2. تبدیل کاربر به Project Definition Actor  
3. تبدیل بازدید → RFQ (Request For Engineering)

---

# 3. اصل بنیادین UX

User → Understanding System → Trust → Engineering Confidence → Project Definition

---

# 4. مدل ذهنی کاربر

کاربر: معمار، EPC، کارفرمای صنعتی، مدیر پروژه

نگرانی‌ها:
1. آیا قابل اجراست؟
2. آیا طراحی قابل تبدیل به اجراست؟
3. آیا ریسک کنترل می‌شود؟
4. آیا تیم مهندسی دارد؟
