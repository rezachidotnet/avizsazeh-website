# AECS Typography System
Version: 1.0
System: Architectural Engineering Communication System (AECS)

---

## 1. Typography Philosophy

Typography in AECS is structural, not decorative.

It acts as:

- Engineering communication layer
- Decision hierarchy system
- Industrial readability framework

Principles:
- Clarity over decoration
- Structure over emotion
- Engineering logic over visual style

---

## 2. Font Stack

### Primary (Persian / Arabic UI)
--font-primary: "Vazirmatn", "IBM Plex Sans Arabic", sans-serif;

Usage:
- UI system
- RFQ engine
- General content

---

### Secondary (English / Technical)
--font-secondary: "Inter", "IBM Plex Sans", sans-serif;

Usage:
- Engineering docs
- Export content
- Technical diagrams

---

### Display Font (Limited)
--font-display: "Montserrat", sans-serif;

Rule:
- Only for hero sections
- Max 5% usage

---

## 3. Type Scale

H1: 48–64px | 700
H2: 32–40px | 600
H3: 24–28px | 600
H4: 18–20px | 600
Body L: 16–18px | 400
Body S: 14px | 400
Caption: 12px | 400

---

## 4. Line Height

H1: 1.1  
H2: 1.2  
H3: 1.3  
Body: 1.6  
Caption: 1.4  

Rule: readability > aesthetics

---

## 5. Letter Spacing

H1: -0.5px  
H2: -0.3px  
H3: -0.2px  
Body: 0px  
Caption: +0.2px  

---

## 6. Direction System

Persian:
direction: rtl;

English:
direction: ltr;

Rule:
- Numbers always LTR
- Units always consistent (mm, m², kN)

---

## 7. Font Weights

300: rare use  
400: body  
500: emphasis  
600: headings  
700: hero  

Rule: no ultra-bold decorative usage

---

## 8. Typography Behavior Rules

Allowed:
- structured text
- technical documentation
- grid-aligned layouts

Not allowed:
- decorative fonts
- emotional typography
- excessive uppercase styling

---

## 9. UI Mapping

Buttons: 14–16px / 500  
Nav: 14px / 500  
Forms: 14–16px / 400–500  

---

## 10. Accessibility

- min size: 14px
- contrast: 4.5:1 minimum
- no low-contrast overlays

---

## 11. Brand Voice Alignment

Engineering → strong hierarchy  
Technical → structured text  
Marketing → minimal  
RFQ → step-based clarity  

---

## 12. Golden Rule

Typography behaves like steel:

> precise, structural, predictable.
