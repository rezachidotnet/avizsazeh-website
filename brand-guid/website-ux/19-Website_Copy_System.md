# Website Copy System (AECS)

## 1. Core Principle
Website copy is not marketing text. It is an engineering communication layer that translates architectural systems into decision-making logic for industrial clients.

---

## 2. Copy Hierarchy

### Level 1: System Definition
Defines what the brand is (not what it sells).

- System language only
- No emotional persuasion
- No adjectives without technical meaning

Example:
"Engineering architectural ceiling systems for industrial and large-scale projects."

---

### Level 2: System Logic
Explains how the system works.

- Structural explanation
- Modular breakdown
- Process clarity

Example:
"From architectural design → structural validation → fabrication → installation."

---

### Level 3: Proof Layer
Evidence of system reliability.

- Projects
- Metrics
- Execution data

Example:
"32-bed military hospital delivered in under 50 days."

---

### Level 4: Decision Layer
Removes risk from the buyer's mind.

- Certifications
- Engineering standards
- Process transparency

Example:
"All systems are pre-validated for seismic and load requirements."

---

### Level 5: Conversion Layer
Direct but non-salesy action.

- RFQ (Request for Quotation)
- Engineering consultation
- Technical review request

Example:
"Submit project specifications for engineering evaluation."

---

## 3. Tone of Voice System

### Allowed
- Engineering language
- Structural clarity
- Technical confidence
- Minimal abstraction

### Not Allowed
- Marketing hype
- Emotional persuasion
- Superlatives without data
- Decorative language

---

## 4. Sentence Structure Rules

- Short declarative sentences
- One idea per line
- Active voice preferred
- Avoid ambiguity

Bad:
"We provide the best innovative solutions for modern ceilings."

Good:
"We design and engineer suspended ceiling systems for industrial environments."

---

## 5. Messaging Blocks

### Hero Block
Define system + category + authority

### System Block
Explain architecture of offering

### Proof Block
Show real execution

### Engineering Block
Show how system is validated

### Conversion Block
Request technical input, not purchase

---

## 6. Keyword Strategy

Primary:
- Engineering ceiling systems
- Architectural envelope systems
- Industrial building systems

Secondary:
- Space optimization systems
- Structural ceiling design
- Modular construction systems

---

## 7. Anti-Marketing Rule

Do NOT:
- Sell
- Persuade emotionally
- Use hype language

DO:
- Explain systems
- Reduce uncertainty
- Enable decision-making

---

## 8. Final Principle

This website does not "convince".

It "eliminates uncertainty in engineering decisions."
